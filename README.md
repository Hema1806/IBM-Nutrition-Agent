# 🥗 AI-Powered Personalized Nutrition Agent using IBM watsonx Orchestrate

## 📌 Project Overview

The **AI-Powered Personalized Nutrition Agent** is an intelligent conversational assistant developed using **IBM watsonx Orchestrate**. It helps users receive personalized nutrition guidance by understanding their health information, dietary preferences, and lifestyle. The agent generates customized meal recommendations and healthy lifestyle suggestions using IBM's foundation model and a curated nutrition knowledge base.

This project demonstrates the practical use of Agentic AI in the healthcare domain to promote healthier eating habits and improve nutritional awareness.

---

## 🎯 Problem Statement

Many people rely on generic diet plans that do not consider personal factors such as age, weight, activity level, food preferences, allergies, or medical conditions. This often leads to unhealthy eating habits and ineffective nutrition planning.

The goal of this project is to develop an AI-powered nutrition assistant that provides personalized diet recommendations through natural language conversations.

---

## 🎯 Objectives

- Develop an AI-powered Nutrition Agent using IBM watsonx Orchestrate.
- Collect user information such as age, weight, activity level, and dietary preferences.
- Generate personalized meal and nutrition plans.
- Provide healthy lifestyle recommendations.
- Answer nutrition-related queries using a knowledge base.
- Promote awareness of balanced nutrition and healthy living.

## 🧠 Knowledge Base

The Nutrition Agent uses uploaded PDF documents as its knowledge source, including:

- Nutrition Guidelines
- BMI & Calorie Guide
- Healthy Lifestyle Guide
- Indian Meal Planner

These documents help the agent generate accurate and context-aware responses.

---

## ⚙️ Workflow

1. User interacts with the Nutrition Agent.
2. The agent collects health and dietary information.
3. User queries are analyzed by the IBM Foundation Model.
4. Relevant information is retrieved from the knowledge base.
5. Personalized nutrition recommendations are generated.
6. The agent provides healthy lifestyle suggestions.
